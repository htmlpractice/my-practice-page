Introduction

Holá! This is the page that I use for practicing typing up HTML documents–with CSS! The CSS file is a separate file to this HTML file you're currently reading in the browser; it tells the browser how the text in the HTML file should be styled.

Text styling & CSS

Here you can see that I have used CSS to make all main headers red and all main paragraphs dark blue by default. As you can see, I can also change the colour of individual words and sentences. I will explain how shortly.

Withhout having to get into CSS, we can use the strong or b elements to make certain words show up in bold, by putting the word inside a tag with the element, like this:

<bold> word </bold>

In the same way we can use em or i to italicize words. Do note, however, that although strong and b have the same effect on the appearance of the text, as do em and i, these elements have different semantics, or meanings.

We can define certain parts of our document by class to customize them however we want to. It's simple: I make an entry for class "c1" in the CSS file, and specify that the font for anything that's class "c1" must be green and 26px in size. Then I go to the HTML and bracket the word I want to change (eg. "green") with an anchor tag and insert the class element, which looks like this:

<a class="c1"> green </a>

Bingo! Big green text!

If you want to do this with a whole paragraph, just insert the class="x" element after the p in the paragraph tag. This is how I set the colour and size of all the font on this page. You can have as many classes as you want.

Mouseover Events

Remember how the two images I used had some text that you could see when you moused over it? We can dig a little deeper into these "mouseover events" and see what else we can do to make our page feel more interactive. Check out this header element:

Mouse Over Me

Can you see the text colour change as you move the mouse cursor over it? We can do it for a normal paragraph too:

Mouse Over Me.

What I've done here is inserted script, or code, into the body of the HTML. For each of those "Mouse Over Me's" I have typed in a few short lines of "function" script that tell the browser what to do when when the mouse cursor is over the text, and when the mouse cursor is not over the text. For example, you can tell the browser to display a specific heading as red normally but to display as black when you mouse over it. Lists

Here's a quick general definition of what lists are:

Lists
